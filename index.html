FinanceTracker
Swift, SwiftUI
FinanceTracker is an app that allows users to flexibly track and categorize transactions, visualize spending patterns, and store data securely on the cloud. Users are greeted with a page that allows them to file a transaction with its name, date, cost, and category. As transaction history builds, users see beautiful visualizations of how their spending changes over time and categorial ratios. On the second page, users can view their sorted transaction history and view each transaction in detail, as well as delete transactions. On the third page, users can integrate a password to their transaction profile by either publishing their current data to the cloud with a new password, or pull down existing transactions with an existing password. This enables users to use FinanceTracker across multiple platforms, as well as multiple profiles on one device. As the transaction history changes, the cloud storage is automatically updated to the current password. Users can also customize their categories and spending limits.
Mesh Generator
Python, Numpy, OpenCV, Scipy
Using structured illumination, I captured dozens of images of a figurine from two different camera angles. I also captured images of a chessboard with known size in order for calibration. Then, I used OpenCV's chessboard corners function to calibrate the extrinsic and intrinsic parameters of the camera. Using the images of the figurine, I wrote algorithms to mask out the background of the image, and used Matplotlib to map out my scene. Then, using the Scipy Delauney algorithm to generate triangles based on the points of each face of the figurine. I implemented manual mesh smoothing algorithms like triangle based pruning, boundary pruning, and point averaging. Finally, using the triangle and color information, I imported the meshes into MeshLab and used Poisson based surface reconstruction and point based mesh gluing algorithms to create a full, 3D, color mesh.
