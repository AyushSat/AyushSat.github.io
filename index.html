Mesh Generator
Python, Numpy, OpenCV, Scipy
Using structured illumination, I captured dozens of images of a figurine from two different camera angles. I also captured images of a chessboard with known size in order for calibration. Then, I used OpenCV's chessboard corners function to calibrate the extrinsic and intrinsic parameters of the camera. Using the images of the figurine, I wrote algorithms to mask out the background of the image, and used Matplotlib to map out my scene. Then, using the Scipy Delauney algorithm to generate triangles based on the points of each face of the figurine. I implemented manual mesh smoothing algorithms like triangle based pruning, boundary pruning, and point averaging. Finally, using the triangle and color information, I imported the meshes into MeshLab and used Poisson based surface reconstruction and point based mesh gluing algorithms to create a full, 3D, color mesh.
